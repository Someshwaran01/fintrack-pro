ℹ️ Manual Setup Instructions

Since API keys have read-only access, please follow these steps:

  1. 📝 Open Your Google Sheet
  2. Click the + button at the bottom to add new tabs
  3. Create 3 tabs with these exact names:
    • Bills
    • Medical
    • Home
  4. In Bills tab, add these headers in row 1:
    ID | Card Name | Category | Due Date | Month | Is EMI | EMI Details | Total Amount | Tenure | Monthly Amount | Payments
  5. In Medical tab, add these headers in row 1:
    ID | Description | Amount | Date | Month | Category | Notes
  6. In Home tab, add these headers in row 1:
    ID | Description | Amount | Date | Month | Category | Notes
  7. Make sure the sheet is shared as "Anyone with the link can view"

Then click the button above to verify the setup!
